3. What is a RESTful API?

  • A) An API that only works with SOAP
  • B) An API that follows the principles of Representational State Transfer (REST)
  • C) An API that does not require authentication
  • D) An API that is only used in mobile applications
Answer: B) An API that follows the principles of Representational State Transfer (REST)

4. What is the primary data format used in RESTful APIs?

  • A) XML
  • B) HTML
  • C) JSON
  • D) CSV
Answer: C) JSON

5. What HTTP method is commonly used to retrieve data from an API?

  • A) POST
  • B) PUT
  • C) GET
  • D) DELETE
Answer: C) GET

6. What HTTP method is used to send data to an API for processing?

  • A) GET
  • B) POST
  • C) DELETE
  • D) PATCH
Answer: B) POST

7. What is an API endpoint?

  • A) A physical server location
  • B) A specific URL where an API receives requests
  • C) The homepage of an API
  • D) A security feature in API authentication
Answer: B) A specific URL where an API receives requests

8. What is an HTTP status code 200?

  • A) The request was successful
  • B) The request failed due to an error
  • C) The request is unauthorized
  • D) The requested resource was not found
Answer: A) The request was successful

9. What does an HTTP status code 404 indicate?

  • A) The request was successful
  • B) The server encountered an internal error
  • C) The requested resource was not found
  • D) The request is unauthorized
Answer: C) The requested resource was not found

10. What does an API key do?

  • A) It encrypts API responses
  • B) It authenticates API requests
  • C) It defines API request limits
  • D) It prevents API rate limiting
Answer: B) It authenticates API requests

11. What is the purpose of an HTTP header in an API request?

  • A) To define the database schema
  • B) To provide additional information about the request or response
  • C) To store large files in API calls
  • D) To increase API request speed
Answer: B) To provide additional information about the request or response

12. What is API rate limiting?

  • A) Restricting the number of API requests a client can make
  • B) Increasing API response time
  • C) Preventing API access for unauthorized users
  • D) Reducing API security features
Answer: A) Restricting the number of API requests a client can make

13. What does an HTTP status code 500 indicate?

  • A) The request was successful
  • B) The server encountered an internal error
  • C) The requested resource was not found
  • D) The request is unauthorized
Answer: B) The server encountered an internal error

14. What is CORS in API communication?

  • A) A security feature that restricts web requests to different domains
  • B) A method for encrypting API responses
  • C) A process for caching API responses
  • D) A strategy for compressing large API payloads
Answer: A) A security feature that restricts web requests to different domains

15. What does an HTTP status code 401 mean?

  • A) The request was successful
  • B) Unauthorized request
  • C) The requested resource was not found
  • D) The server is down
Answer: B) Unauthorized request

16. What is an API response?

  • A) The data sent back from the server after an API request
  • B) A request made by the user
  • C) A JavaScript function
  • D) A new API endpoint
Answer: A) The data sent back from the server after an API request

17. What does an HTTP status code 201 mean?

  • A) The request was successful, and a new resource was created
  • B) The request failed due to an internal server error
  • C) The request was unauthorized
  • D) The request returned no data
Answer: A) The request was successful, and a new resource was created

  • B) A method for securing API calls
  • C) A component that runs the API server
  • D) A type of API authentication
Answer: A) A key-value pair appended to a URL to modify the request

  • B) A method for storing API responses
  • C) A system for preventing API rate limiting
  • D) A way to structure API requests

20. What is an example of an API client?

21. What does an HTTP 403 status code indicate?

22. What is an API timeout?

30. What is the difference between synchronous and asynchronous requests?
